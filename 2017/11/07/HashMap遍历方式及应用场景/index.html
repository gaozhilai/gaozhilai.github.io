HashMap遍历方式及应用场景

2017-11-07

HashMap可以通过entrySet()方法获取到EntrySet,然后用iterator或者增强for循环遍历:
效率高,以后一定要使用此种方式！

获取keySet,也可以用iterator或是增强for循环遍历key,然后通过map自带方法取值:

Map map = new HashMap();
Iterator iter = map.keySet().iterator();
while (iter.hasNext()) {
Object key = iter.next();
Object val = map.get(key);
}
//增强for循环方式:
for(Object key:map.keySet()){
Object value=map.get(key);
}

最后一种自残的遍历方式,只能遍历值,通过values()方法获取map的值集合,也可以增强for循环或是迭代器遍历

Map map = new HashMap();
Iterator iter=map.values().iterator();
while(iter.hasNext()){
Object value=iter.next();
}
//增强for循环方式
for(Object val:map.values()){
Object value=val;
}

####结论:
遍历HashMap键值方式有两种,两种每种都可以分为迭代器和增强for循环方式遍历,还有一种遍历方式还有一种遍历方式只能遍历值,此种方式也分为迭代器和增强for循环.
