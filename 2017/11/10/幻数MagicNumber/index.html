幻数MagicNumber

2017-11-10 • 标签: 娱乐, 趣闻

在学习dicom的过程中了解到了一个概念，MagicNumer(幻数).

###概念：
幻数就是程序中让人不能看出含义的对常量的赋值.比如在一段程序中:

int magicNo = 3721; // what the fuck 3721 mean? so it's a magic number

而表意清楚就不是幻数了,比如:

const size_t arrayLen = 3721;  // given 3721 the meaning of the array length
for (size_t i = 0; i < arrayLen; i++) ...

###例子:
最著名的例子属于 約翰·卡馬克 （John D. Carmack）在速算平方根函数Q_rsqrt中的幻数0x5f3759df了,人们普遍认为这个幻数是卡马克发明引入的,单被本人否认,真正来源是个谜…

float Q_rsqrt( float number )
{
    long i;
    float x2, y;
    const float threehalfs = 1.5F;

    x2 = number * 0.5F;
    y  = number;
    i  = * ( long * ) &y;                       // evil floating point bit level hacking（对浮点数的邪恶位级hack）
    i  = 0x5f3759df - ( i >> 1 );               // what the fuck?（这他妈的是怎么回事？）
    y  = * ( float * ) &i;
    y  = y * ( threehalfs - ( x2 * y * y ) );   // 1st iteration （第一次牛顿迭代）
//      y  = y * ( threehalfs - ( x2 * y * y ) );   // 2nd iteration, this can be removed（第二次迭代，可以删除）

    return y;
}

###应用:

幻数可以用来判断文件格式,可以判断文件是否损坏,Linux下的file命令就是通过MagicNumber判断文件类型的,比如java编译后产生的Class文件用来表明文件类型的MagicNumber就是0xCAFEBABE.
