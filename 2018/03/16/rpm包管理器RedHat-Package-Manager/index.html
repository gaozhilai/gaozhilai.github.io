rpm包管理器RedHat Package Manager

2018-03-16 • 标签: linux

RedHat Package Manager

今天总结一下rpm包管理器(RedHat Package Manager)的常用命令及其含义.

敲黑板划重点

经常用到的rpm安装命令是
安装rpm包并显示信息进度

rpm -ivh 安装包全名

man中对应的参数的解释:
-i,–install
-h, –hash
Print 50 hash marks as the package archive is unpacked. Use with
-v|–verbose for a nicer display.

查看所有已经安装的rpm包

rpm -qa

如果已经安装的包太多可以用下面命令找出已经安装的含有关键字的包名

rpm -qa|grep 关键字

查看已经安装的rpm包信息
列出包安装所有文件位置:

rpm -ql 包名 ##包名可能与文件名并不相同,要通过上面命令查出包名

列出已安装包的信息:

rpm -qi 包名

参数含义:
-l, –list
List files in package.
-i, –info
Display package information, including name, version, and
description. This uses the –queryformat if one was specified.
